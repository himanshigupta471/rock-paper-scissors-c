# Rock Paper Scissors Game (C)

This is a simple command-line Rock Paper Scissors game developed using C language.

## Features
- User vs Computer gameplay
- Random computer choice
- Multiple rounds
- Score tracking

## How to Run
1. Compile the code:
   gcc project.c -o project

2. Run the program:
   ./project

## Skills Used
- C Programming
- Logic Building
- User Input Handling
